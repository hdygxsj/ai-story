# AI Story Novel Character Entrance

Atomic skill for making major character entrances vivid, anchored, and story-useful.

## Setup

Use AI Story through the Go CLI and platform data:

```bash
ai-story agent manifest
ai-story api request GET /novels/{novel_id}/nodes
ai-story api request GET /novels/{novel_id}/creative-assets
ai-story api request GET /novels/{novel_id}/character-states
ai-story api request GET /novels/{novel_id}/relationship-edges
```

## Hard Rule

Core characters cannot enter naked: a heroine, love interest, rival, villain, or recurring ally needs role, scene relevance, first impression, and at least one external marker the reader can remember.

## Entrance Layers

Check role, relevance, first impression, appearance, clothing, behavior, response, and agency. Clothing should imply class, restraint, secrecy, vulnerability, rebellion, daily discipline, or pressure. Avoid inventory-like outfit lists.

When beauty matters, make it do story work. Use one suggestive detail, a behavioral contrast, or another character's reaction instead of stacking labels such as "校花", "绝美", or "身材很好". Keep school-age scenes restrained and age-appropriate.

## Repair Order

Clarify role, then relationship, then first impression, then one memorable visual or behavioral marker. Do not patch weak entrance by dumping biography.
