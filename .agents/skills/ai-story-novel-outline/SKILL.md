# AI Story Novel Outline

Atomic skill for building a Platform outline in AI Story records.

Use `ai-story agent manifest`, then read platform nodes, creative assets, timeline events, and memory. Build story engine, volume arc, six-chapter pack, and chapter outline. Each outline decision should say whether it is advice only or durable canon.

Apply accepted outline facts with timeline, memory, character, and world/material tools. Keep outlines tied to platform state rather than a separate file workflow.
